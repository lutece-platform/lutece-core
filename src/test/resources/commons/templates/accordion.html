<@h2># Parameters documentation <@h3>accordionContainer

The container for the panel, header, body elements of the accordion.

<@table> <@tr> <@th>Parameter <@th>Value <@th>Default <@th>Use <@tr> <@th>id <@td>string <@td>none <@td>Unique identifier for the accordion container. If not provided, a default ID will be generated. <@h3>accordionPanel

Defines a panel in the accordion. The panel can be collapsed or expanded.

<@table> <@tr> <@th>Parameter <@th>Value <@th>Default <@th>Use <@tr> <@th>color <@td>string <@td>none <@td>Defines the color of the panel. Options include primary, info, warning, danger, success. <@tr> <@th>collapsed <@td>boolean <@td>true <@td>Indicates whether the panel is collapsed by default. <@tr> <@th>childId* <@td>string <@td>none <@td>Unique identifier for the child element of the accordion panel. If not provided, a default ID will be generated. <@tr> <@th>id <@td>string <@td>none <@td>Unique identifier for the accordion panel. If not provided, a default ID will be generated. <@tr> <@th>params <@td>string <@td>none <@td>Additional parameters for the panel. <@h3>accordionHeader

Defines the header of the accordion panel. It can include a title and an icon.

<@table> <@tr> <@th>Parameter <@th>Value <@th>Default <@th>Use <@tr> <@th>id <@td>string <@td>none <@td>Unique identifier for the accordion header. If not provided, a default ID will be generated. <@tr> <@th>title <@td>string <@td>none <@td>H2 title of the accordion header. <@tr> <@th>parentId* <@td>string <@td>none <@td>Variable from the accordionContainer macro <@tr> <@th>childId* <@td>string <@td>none <@td>Variable from the accordionPanel macro <@tr> <@th>boxTools <@td>boolean <@td>false <@td>Indicates whether the box tools are displayed in the header. <@tr> <@th>params <@td>string <@td>none <@td>Additional parameters for the header. <@tr> <@th>headerIcon <@td>string <@td>none <@td>Icon to display in the header. <@h3>accordionBody

Defines the body of the accordion panel. It contains the content displayed when the panel is expanded.

<@table> <@tr> <@th>Parameter <@th>Value <@th>Default <@th>Use <@tr> <@th>id <@td>string <@td>none <@td>The childID defined in the parent accordionPanel macro. <@tr> <@th>class <@td>string <@td>none <@td>The childClass defined in the parent accordionPanel macro. <@tr> <@th>expanded <@td>boolean <@td>none <@td>The expanded state defined in the parent accordionPanel macro. <@tr> <@th>params <@td>string <@td>none <@td>Additional parameters for the body. <@footer />
